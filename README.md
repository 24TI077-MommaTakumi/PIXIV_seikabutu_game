# PIXIV_seikabutu_game
大学で所属しているサークル活動で作成したゲームです。UnityとVisualStudioを使用して作成しました。（プログラミング言語：C#）
音声素材はフリー素材を用いて、それ以外は全て自分で作成しました。（ドット絵素材、プログラミングコードなど）


ゲーム内容については以下の通りです。

タイトル：「スーパーマリモ」
ジャンル：2Dアクション
概要：
ステージ制の2Dアクションゲームです。（全5ステージ）
基本操作は左右移動とジャンプのみですが、壁を上ることができます。
初代スーパーマリオブラザーズをモチーフに、マリオと語感が似ているマリモを主人公にしてこのゲームを作成しました。
